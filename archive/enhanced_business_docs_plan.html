HelpmApp: Enhanced Business Documentation (Reset Edition) ​

Version: 2.0
Last Updated: 17 Nov 2025

This guide explains how to navigate the refreshed documentation suite after the deliberate clean-slate rebuild. Every referenced file now reflects the new direction, metrics, and operating assumptions.

When to Use Each Document ​

  • Grant / funder outreach: Start with Executive Summary + Quick Reference; attach Market Analysis for need, Financial Projections for budget, Impact Framework for evaluation methodology.
  • Partnership development: Pair Quick Reference with Operational Plan milestones; include Market Analysis sections tailored to provider/local authority/corporate audience.
  • Internal planning: Operational Plan + Financial Projections provide execution guardrails; Impact Framework ensures measurement discipline.
  • Press / storytelling: Pull headline numbers from Quick Reference; use Executive Summary for narrative context.

Versioning & Governance ​

  • All primary docs now track Version 2.0 (Nov 2025) to signify alignment with the rebuild.
  • Superseded drafts were archived in docs/_archive/2025-11-17-restructure/ for historical reference.
  • Update cadence: review quarterly or when major strategic shifts occur (e.g., new funding, geographic expansion).
  • Owner assignments:
    • Executive Summary / Quick Reference: Founders
    • Market & Operations: Partnerships + Product
    • Finance: Finance lead / fractional CFO
    • Impact: Impact & Data team

Key Data Sources (Validated for Reset) ​

  • Trust for London, Crisis UK, CHAIN, Institute for Government, GLA borough stats (homelessness data).
  • London Homelessness Services Atlas, borough open data portals (service inventory).
  • Academic research on digital inclusion + gamification for vulnerable populations.
  • Legacy HelpmApp analytics (archived) to inform baseline assumptions.

Using This Guide ​

  1. Confirm which stakeholder you are preparing for; select documents per table above.
  2. Cite version numbers when sharing externally to avoid confusion with archived materials.
  3. Feed new insights back into the relevant doc owners to keep content living and accurate.

Document Versions ​

All documents in this package are Version 2.0, last updated November 2025. As HelpmApp evolves and new data becomes available, these documents should be reviewed and updated quarterly to maintain accuracy and relevance.

Next Steps ​

With this comprehensive documentation package, HelpmApp is well-positioned to:

  1. Secure Funding: Apply for grants from charitable trusts, approach impact investors, and pitch to angel investors or venture philanthropists

  2. Develop Partnerships: Engage service providers with clear value propositions, approach local authorities with data-driven proposals, and recruit corporate sponsors for the rewards program

  3. Execute Operations: Follow the operational plan roadmap, track progress against financial projections, and measure impact using the established framework

  4. Communicate Impact: Report to stakeholders using consistent metrics and narratives, share user stories and case studies, and contribute to sector knowledge through publications and presentations

  5. Iterate and Improve: Continuously refine the business model based on data and feedback, adapt to changing market conditions and user needs, and scale successful approaches to new geographies
